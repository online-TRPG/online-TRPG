# 직업 번역 검수 기준

> 목적: 직업 문서가 캐릭터 생성과 레벨업에 필요한 정보를 빠뜨리지 않도록 확인하기 위한 체크리스트.

## 공통 필수 항목

각 직업 문서에는 다음 항목이 있어야 한다.

- 원문명과 SRD 대표 서브클래스/선택지
- 원문 페이지 위치
- 히트 다이스, 1레벨 HP, 레벨 상승 HP
- 방어구, 무기, 도구 숙련
- 내성 굴림 숙련
- 선택 가능한 기술 숙련
- 시작 장비
- 1-20레벨 전체 진행표
- 직업별 핵심 자원 표기
- SRD 대표 서브클래스 기능
- DB 분리 메모

## 주문시전자 필수 항목

마법을 시전할 수 있는 직업 문서에는 다음 항목이 추가로 있어야 한다.

- 주문시전 능력치
- 주문 내성 DC 계산식
- 주문 공격 수정치 계산식
- 캔트립 수 또는 캔트립 진행표
- 알고 있는 주문 수 또는 준비 주문 수 공식
- 레벨별 주문 슬롯 표
- 의식 시전 가능 여부
- 주문시전 초점구
- 권역/맹세/후원자/서클 등으로 추가되는 주문 목록

## 직업별 추가 확인 항목

| 직업 | 반드시 확인할 고유 항목 |
|---|---|
| 바바리안 | 격노 횟수, 격노 피해, 원초적 길 기능 |
| 바드 | 캔트립, 알고 있는 주문, 1-9레벨 슬롯, 바드의 고양감 주사위 |
| 클레릭 | 캔트립, 준비 주문 공식, 1-9레벨 슬롯, 권역 주문, 신성 변환 |
| 드루이드 | 캔트립, 준비 주문 공식, 1-9레벨 슬롯, 야생 변신 제한, 대지의 서클 주문 |
| 파이터 | 추가 공격 횟수, 행동 연쇄 횟수, 불굴 횟수, 무예 아키타입 |
| 몽크 | 무술 주사위, 기 점수, 비무장 이동 보너스, 수도 전통 |
| 팔라딘 | 준비 주문 공식, 1-5레벨 슬롯, 맹세 주문, 신성한 강타 |
| 레인저 | 알고 있는 주문, 1-5레벨 슬롯, 숙적, 자연 탐험가, 사냥꾼 선택지 |
| 로그 | 암습 피해, 전문화, 교활한 행동, 로그 아키타입 |
| 소서러 | 캔트립, 알고 있는 주문, 1-9레벨 슬롯, 소서리 포인트, 메타매직 |
| 워락 | 캔트립, 알고 있는 주문, 계약 슬롯 수와 슬롯 레벨, 영창 수, 섬뜩한 영창 목록, 신비의 비밀, 확장 주문 |
| 위저드 | 캔트립, 준비 주문 공식, 주문책 규칙, 1-9레벨 슬롯, 비전 전통 |

## 현재 재검토 결과

- 12개 직업 문서에 `전체 레벨 진행표`를 추가했다.
- 주문시전자 8개 직업에 주문 슬롯 또는 계약 슬롯 진행표를 추가했다.
- 바드, 소서러, 워락, 위저드에는 캔트립/알고 있는 주문 진행표를 추가했다.
- 클레릭, 드루이드, 팔라딘에는 준비 주문 공식과 추가 주문 목록을 추가했다.
- 레인저에는 알고 있는 주문과 1-5레벨 주문 슬롯 진행표를 추가했다.
- 바바리안, 몽크, 로그에는 각각 격노/기/암습 진행표를 추가했다.
- 소서러에는 유연한 주문시전 변환표와 메타매직 비용/효과 표를 추가했다.
- 워락에는 SRD 섬뜩한 영창 목록과 선결 조건/효과 요약을 추가했다.
