# 능력 판정과 d20 규칙

> 상태: 1차 번역/정리. 실제 플레이 중 판정 요청과 난이도 산정에 필요한 규칙 중심.

## 포함 범위

## 6개 능력치

| 능력치 | 한국어 | 의미 |
|---|---|---|
| Strength | 근력 | 신체적 힘 |
| Dexterity | 민첩 | 민첩성, 반사신경, 균형 |
| Constitution | 건강 | 지구력 |
| Intelligence | 지능 | 추론과 기억 |
| Wisdom | 지혜 | 감각과 통찰 |
| Charisma | 매력 | 인격의 힘, 존재감 |

게임의 세 가지 주요 굴림인 능력 판정, 내성 굴림, 공격 굴림은 모두 6개 능력치에 의존한다.

```text
d20 + 관련 능력 수정치 + 적용되는 보너스
```

## 능력치와 수정치

능력치 10 또는 11은 일반적인 인간 평균이다. 모험가와 많은 몬스터는 대부분의 능력치에서 평균보다 뛰어나다. 보통 사람은 18에 도달하는 것이 최고 수준이고, 모험가는 20까지 도달할 수 있다. 몬스터나 신적 존재는 30까지 도달할 수 있다.

| 능력치 | 수정치 |
|---:|---:|
| 1 | -5 |
| 2-3 | -4 |
| 4-5 | -3 |
| 6-7 | -2 |
| 8-9 | -1 |
| 10-11 | +0 |
| 12-13 | +1 |
| 14-15 | +2 |
| 16-17 | +3 |
| 18-19 | +4 |
| 20-21 | +5 |
| 22-23 | +6 |
| 24-25 | +7 |
| 26-27 | +8 |
| 28-29 | +9 |
| 30 | +10 |

표를 보지 않고 계산하려면 능력치에서 10을 뺀 뒤 2로 나누고 내림한다.

```text
능력 수정치 = floor((능력치 - 10) / 2)
```

## 유리함과 불리함

특수 능력, 주문, 상황은 능력 판정, 내성 굴림, 공격 굴림에 유리함(Advantage) 또는 불리함(Disadvantage)을 줄 수 있다.

- 유리함: d20 두 개를 굴리고 높은 값을 사용한다.
- 불리함: d20 두 개를 굴리고 낮은 값을 사용한다.
- 여러 유리함이 있어도 추가 d20은 하나만 굴린다.
- 여러 불리함이 있어도 추가 d20은 하나만 굴린다.
- 유리함과 불리함이 동시에 있으면 서로 상쇄되어 d20 하나만 굴린다.
- 상쇄는 개수와 상관없이 적용된다. 유리함이 여러 개이고 불리함이 하나여도 d20 하나만 굴린다.

유리함 또는 불리함이 있을 때 하플링의 행운처럼 d20을 다시 굴리는 규칙이 적용되면, 두 d20 중 하나만 다시 굴릴 수 있고 어느 주사위를 다시 굴릴지 선택한다.

GM은 상황이 판정에 유리하거나 불리하다고 판단하면 유리함 또는 불리함을 부여할 수 있다. 영감(Inspiration)도 유리함을 줄 수 있다.

## 숙련 보너스

캐릭터의 숙련 보너스는 레벨로 결정된다. 몬스터의 숙련 보너스는 스탯블록에 이미 반영되어 있다.

숙련 보너스는 능력 판정, 내성 굴림, 공격 굴림에 쓰인다.

- 같은 굴림이나 수치에 숙련 보너스를 한 번보다 많이 더할 수 없다.
- 어떤 기능이 숙련 보너스를 두 배로 하거나 절반으로 하게 할 수 있다.
- 숙련 보너스를 더하지 않는 판정에서 “두 배”를 적용해도 0을 두 배로 하는 것이므로 보너스가 생기지 않는다.
- 일반적으로 공격 굴림과 내성 굴림에서는 숙련 보너스를 곱하거나 나누지 않는다. 예외가 있으면 해당 기능 설명을 따른다.

## 능력 판정

능력 판정은 캐릭터나 몬스터가 실패 가능성이 있는 행동으로 도전을 극복하려 할 때 사용한다.

GM은 다음을 결정한다.

- 어떤 능력치가 관련되는가?
- 어떤 기술 숙련이 적용될 수 있는가?
- 난이도 DC는 얼마인가?

```text
능력 판정 = d20 + 관련 능력 수정치 + 숙련 보너스(해당할 때)
```

결과가 DC 이상이면 성공한다. 실패하면 목표를 향한 진전이 없거나, GM이 정하는 대가와 함께 제한적으로 진전할 수 있다.

## 일반적인 난이도 DC

| 난이도 | DC |
|---|---:|
| 매우 쉬움 | 5 |
| 쉬움 | 10 |
| 보통 | 15 |
| 어려움 | 20 |
| 매우 어려움 | 25 |
| 거의 불가능 | 30 |

## 대결 판정

한 캐릭터나 몬스터의 행동이 다른 존재의 행동과 직접 맞설 때 대결 판정(Contest)을 사용한다.

예시:

- 두 캐릭터가 바닥에 떨어진 마법 반지를 먼저 낚아채려 한다.
- 몬스터가 문을 밀어 열려 하고, 모험가가 안쪽에서 문을 닫아 버티려 한다.

대결 판정에서는 양쪽이 상황에 맞는 능력 판정을 한다. DC와 비교하지 않고 두 결과를 서로 비교한다. 더 높은 결과가 이긴다. 동률이면 상황은 이전과 그대로 유지된다.

## 기술

기술은 특정 능력치의 세부 사용 분야를 나타낸다.

| 능력치 | 관련 기술 |
|---|---|
| 근력 | 운동 |
| 민첩 | 곡예, 손재주, 은신 |
| 건강 | 관련 기술 없음 |
| 지능 | 비전학, 역사, 조사, 자연, 종교 |
| 지혜 | 동물 조련, 통찰, 의학, 감지, 생존 |
| 매력 | 기만, 위협, 공연, 설득 |

GM은 `지혜(감지) 판정`처럼 특정 능력치와 기술을 함께 요구할 수 있다. 기술 숙련이 있으면 해당 판정에 숙련 보너스를 더한다.

## 다른 능력치와 기술 조합

일반적으로 기술 숙련은 특정 능력치 판정에 적용된다. 하지만 상황에 따라 다른 능력치와 조합할 수 있다.

예시:

- 오래 헤엄쳐야 할 때 `건강(운동)` 판정
- 순수한 완력으로 상대를 겁주려 할 때 `근력(위협)` 판정

이런 조합은 GM이 요청하거나, 플레이어가 적용 가능성을 제안할 수 있다.

## 수동 판정

수동 판정은 d20을 굴리지 않는 특수한 능력 판정이다. 반복 작업의 평균 결과를 나타내거나, GM이 플레이어에게 굴림을 알리지 않고 비밀리에 성공 여부를 판단할 때 사용한다.

```text
수동 판정 점수 = 10 + 해당 판정에 적용되는 모든 수정치
```

- 유리함이 있으면 +5
- 불리함이 있으면 -5

예를 들어 지혜 15이고 감지에 숙련된 1레벨 캐릭터의 수동 지혜(감지)는 14다.

## 협력

두 명 이상의 캐릭터가 함께 작업할 수 있다. 주도하는 캐릭터 또는 가장 높은 능력 수정치를 가진 캐릭터가 유리함을 받고 능력 판정을 한다.

협력 조건:

- 도와주는 캐릭터도 혼자서 그 작업을 시도할 수 있어야 한다.
- 함께하는 것이 실제로 도움이 되는 작업이어야 한다.
- 전투 중에는 도움(Help) 행동이 필요하다.

자물쇠 따기처럼 도둑 도구 숙련이 필요한 작업은, 도둑 도구 숙련이 없는 캐릭터가 도울 수 없다.

## 그룹 판정

여러 캐릭터가 하나의 그룹으로 같은 일을 시도할 때 GM은 그룹 능력 판정을 요구할 수 있다.

- 그룹의 모든 캐릭터가 판정한다.
- 절반 이상이 성공하면 그룹 전체가 성공한다.
- 절반 미만이 성공하면 그룹 전체가 실패한다.

그룹 판정은 늪지대 항해처럼 성공하거나 실패할 때 모두가 함께 영향을 받는 상황에 특히 유용하다.

## 능력치별 대표 사용처

| 능력치 | 대표 판정 | 공격/피해 관련 |
|---|---|---|
| 근력 | 들어 올리기, 밀기, 당기기, 부수기, 등반, 점프, 수영 | 근접 무기 공격과 피해 |
| 민첩 | 균형 잡기, 곡예, 손재주, 은신, 자물쇠 따기, 함정 해제 | 원거리 무기 공격과 피해, 기교 무기 |
| 건강 | 숨 참기, 질병/독 견디기, 장시간 버티기 | 보통 내성 굴림과 HP에 중요 |
| 지능 | 기억, 추론, 조사, 마법/역사/자연/종교 지식 | 일부 주문시전 능력치 |
| 지혜 | 감지, 통찰, 생존, 동물 다루기, 의학 | 일부 주문시전 능력치 |
| 매력 | 설득, 기만, 위협, 공연, 사회적 영향력 | 일부 주문시전 능력치 |

## 운반, 밀기, 끌기

근력 점수는 캐릭터가 들거나 옮길 수 있는 무게를 정한다.

```text
운반 용량 = 근력 점수 x 15 파운드
밀기/끌기/들기 한계 = 근력 점수 x 30 파운드
```

운반 용량을 초과하는 무게를 밀거나 끌면 이동속도는 5피트가 된다.

중형보다 큰 크리처는 크기 단계가 하나 커질 때마다 운반 용량과 밀기/끌기/들기 한계가 두 배가 된다. 초소형(Tiny) 크리처는 이 수치가 절반이 된다.

## 선택 규칙: 하중

하중 변형 규칙을 사용하면 장비 무게가 이동에 더 직접적으로 영향을 준다.

| 조건 | 효과 |
|---|---|
| 근력 점수 x 5 파운드 초과 | 하중 상태. 이동속도 -10피트 |
| 근력 점수 x 10 파운드 초과, 최대 운반 용량 이하 | 중하중 상태. 이동속도 -20피트, 근력/민첩/건강 기반 능력 판정, 공격 굴림, 내성 굴림에 불리함 |

이 선택 규칙을 사용할 때는 방어구 표의 Strength 항목을 무시한다.
