# 종족 번역 검수 기준

> 목적: 종족 문서가 캐릭터 생성에 필요한 종족/하위 종족 특성을 빠뜨리지 않도록 확인하기 위한 체크리스트.

## 공통 필수 항목

각 종족 문서에는 다음 항목이 있어야 한다.

- 원문명
- 원문 페이지 위치
- 능력치 증가
- 나이
- 성향
- 크기
- 이동속도
- 언어
- 종족 고유 특성
- 하위 종족이 있으면 하위 종족 특성
- DB 분리 메모

## 종족별 확인 항목

| 종족 | 반드시 확인할 고유 항목 |
|---|---|
| 드워프 | 암시야, 독 내성 유리함, 독 피해 저항, 전투 훈련, 도구 숙련, 석공의 지식, 언덕 드워프의 지혜 증가와 HP 증가 |
| 엘프 | 암시야, 감지 숙련, 요정 혈통, 무아지경, 하이 엘프의 지능 증가, 무기 훈련, 위저드 캔트립, 추가 언어 |
| 하플링 | 행운, 용감함, 하플링의 민첩함, 라이트풋 하플링의 매력 증가와 타고난 은신성 |
| 인간 | 모든 능력치 +1, 추가 언어 |
| 드래곤본 | 드래곤 혈통 표, 숨결 무기 범위/내성/DC/피해 성장, 피해 저항 |
| 노움 | 암시야, 마법에 대한 지능/지혜/매력 내성 유리함, 바위 노움의 건강 증가, 장인의 지식, 땜장이 장치 3종 |
| 하프엘프 | 매력 +2, 선택 능력치 2개 +1, 암시야, 요정 혈통, 기술 2개 숙련, 추가 언어 |
| 하프오크 | 근력 +2, 건강 +1, 암시야, 위협 숙련, 끈질긴 인내, 야만적 공격 |
| 티플링 | 지능 +1, 매력 +2, 암시야, 화염 저항, `thaumaturgy`, 3레벨 `hellish rebuke`, 5레벨 `darkness`, 매력 주문시전 능력치 |

## 재검토 결과

- 종족별 기계적 특성은 9개 종족 문서에 모두 포함되어 있음을 확인했다.
- 원문 p.3의 공통 종족 특성 설명이 별도 문서로 없어서 `종족_공통_규칙.md`를 추가했다.
- `종족_검수_기준.md`를 추가해 앞으로 종족 문서를 확장하거나 수정할 때 확인할 항목을 고정했다.
