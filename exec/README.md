# exec — 포팅 매뉴얼 산출물

본 폴더는 SSAFY 공통 프로젝트 제출 규정에 따른 **포팅 매뉴얼** 산출물입니다.
GitLab 소스 클론 이후 빌드·배포가 가능하도록 환경 정보와 시연 자료를 정리했습니다.

> 폴더명은 규정에 따라 정확히 `exec` 로 생성했으며, 소스코드는 별도로 업로드하지 않습니다.

## 구성

| 파일 | 내용 | 규정 항목 |
|---|---|---|
| `1_빌드_및_배포_매뉴얼.md` | JVM/웹서버/WAS·버전·환경변수·빌드/배포 절차·특이사항·DB 접속 정보 | 1 |
| `2_외부서비스_명세서.md` | 소셜 인증·AI·스토리지 등 외부 서비스 가입 및 활용 정보 | 2 |
| `3_DB덤프_가이드.md` | DB 덤프 파일 최신본 및 추출/복원 절차 | 3 |
| `4_시연_시나리오.md` | 시연 순서에 따른 화면별·실행별 상세 설명 | 4 |
| `dump/` | DB 덤프 파일(`.dump`) 보관 위치 | 3 |
