# 2. 외부 서비스 명세서

프로젝트에서 사용하는 외부 서비스의 가입 및 활용에 필요한 정보를 정리한 문서입니다.

> ⚠️ 실제 API 키·시크릿·계정 비밀번호는 보안상 본 문서에 기재하지 않습니다.
> 운영 값은 Jenkins Credentials(`env-backend`, `env-ai`) 또는 팀 보안 채널을 통해 전달받으세요.

---

## 2.1 Google AI Studio (AI GM 엔진)

| 항목 | 내용 |
|---|---|
| 용도 | AI 게임 마스터 — 상황 해석(interpreter), 서술(narrator), 연출(director), 요약, NPC 대사 생성 |
| 서비스 | Google AI Studio (Gemini API) |
| 사용 모델 | `gemma-4-31b-it` |
| 가입 / 발급 | https://aistudio.google.com → API key 발급 |
| 사용 SDK | `google-genai` (Python, 1.10+) |
| 연동 위치 | `ai/` (FastAPI AI 서버) |
| 필요 환경변수 (`.env.ai`) | `GOOGLE_API_KEY`, `AI_PROVIDER=google-ai-studio`, `AI_MODEL_DEFAULT` 등 |
| 비고 | 백엔드는 AI 서버를 프록시로 호출(`AI_SERVICE_URL`), AI 서버만 Google API 와 직접 통신 |

## 2.2 Kakao 소셜 로그인 (OAuth 2.0)

| 항목 | 내용 |
|---|---|
| 용도 | 카카오 계정 기반 회원가입 / 로그인 |
| 가입 / 발급 | https://developers.kakao.com → 애플리케이션 등록 |
| 발급 정보 | REST API 키, Client Secret |
| Redirect URI | `https://k14a201.p.ssafy.io/<카카오 콜백 경로>` 등록 필요 |
| 인증 엔드포인트 | `https://kauth.kakao.com/oauth/authorize`, `.../oauth/token` |
| 연동 위치 | `be/src/modules/users/users.service.ts` |
| 필요 환경변수 (`.env.backend`) | `KAKAO_REST_API_KEY`, `KAKAO_CLIENT_SECRET` |
| 비고 | 카카오 개발자 콘솔에서 **카카오 로그인 활성화** 및 Redirect URI 등록 필수 |

## 2.3 Discord 소셜 로그인 (OAuth 2.0)

| 항목 | 내용 |
|---|---|
| 용도 | Discord 계정 기반 회원가입 / 로그인 |
| 가입 / 발급 | https://discord.com/developers/applications → New Application |
| 발급 정보 | Client ID, Client Secret |
| Redirect URI | OAuth2 설정에서 `https://k14a201.p.ssafy.io/<디스코드 콜백 경로>` 등록 필요 |
| 연동 위치 | `be/src/modules/users/users.service.ts` |
| 필요 환경변수 (`.env.backend`) | `DISCORD_CLIENT_ID`, `DISCORD_CLIENT_SECRET` |
| 비고 | OAuth2 Scope: `identify` (및 필요 시 `email`) |

## 2.4 Cloudflare R2 (오브젝트 스토리지)

| 항목 | 내용 |
|---|---|
| 용도 | 시나리오 노드 이미지 · VTT 맵 이미지 업로드 / 보관 |
| 서비스 | Cloudflare R2 (S3 호환 오브젝트 스토리지) |
| 가입 / 발급 | https://dash.cloudflare.com → R2 → 버킷 생성, API 토큰(Access Key) 발급 |
| 발급 정보 | Account ID, Bucket Name, Access Key ID, Secret Access Key, 공개 도메인 |
| 연동 위치 | `be/src/modules/scenarios/scenarios.service.ts` |
| 필요 환경변수 (`.env.backend`) | `R2_ACCOUNT_ID`, `R2_BUCKET_NAME`, `R2_ACCESS_KEY_ID`, `R2_SECRET_ACCESS_KEY`, `R2_PUBLIC_BASE_URL`, `R2_MAX_IMAGE_BYTES`, `R2_MAX_MAP_IMAGE_BYTES` |
| 비고 | 공개 접근용 버킷/도메인 설정 필요. 업로드 크기 제한: 이미지 5MB / 맵 10MB |

## 2.5 Let's Encrypt (SSL 인증서)

| 항목 | 내용 |
|---|---|
| 용도 | HTTPS 서비스용 SSL/TLS 인증서 발급 및 자동 갱신 |
| 서비스 | Let's Encrypt (무료 인증기관) / Certbot |
| 비용 | 무료 (별도 가입 불필요) |
| 대상 도메인 | `k14a201.p.ssafy.io` |
| 연동 위치 | `docker-compose.yml` 의 `certbot` 서비스, `infra/nginx/` |
| 비고 | 최초 발급만 수동 1회, 이후 12시간 주기 자동 갱신 |
